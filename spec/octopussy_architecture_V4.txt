1. VISION &amp; POSITIONING
1.1 What Is Octopussy?
Octopussy is a production-grade, open-source, agentic AI platform built entirely on Capability-Centric Architecture (CCA) 0.2. It enables individuals, teams, and organizations to deploy autonomous AI agents — individually or in coordinated multi-agent teams — across any platform: macOS, Windows, Linux, single-board computers (Raspberry Pi, Jetson Nano), Docker containers, and Kubernetes clusters.
Octopussy is the operating system for autonomous AI agents: it handles agent lifecycle, LLM routing, memory, RAG, security, sandboxing, token budgets, scheduling, multi-agent coordination, observability, and extensibility — so developers focus exclusively on agent goals and behaviors.
Installation is intentionally frictionless:
pip install octopussy
octopussy install
octopussy start
octopussy status

1.2 Design Philosophy
Seven principles govern every design decision in Octopussy. These are enforced invariants, not aspirational guidelines. Any version of the code that violates one is defective by definition.

2. ARCHITECTURAL FOUNDATIONS
2.1 Capability-Centric Architecture (CCA 0.2) Application
Every functional unit of Octopussy is a Capability structured as:

Capability Nucleus: three mandatory layers
Essence — pure domain logic; no infrastructure, no I/O, no external dependencies
Realization — technical mechanisms: database adapters, queue implementations, HTTP clients
Adaptation — external interfaces: REST adapters, gRPC adapters, CLI adapters, MCP adapters


Capability Contract: formal specification of
Provisions — what the Capability provides to others (its public interface)
Requirements — what the Capability needs from others (its dependencies)
Protocols — interaction rules, ordering constraints, invariants


Evolution Envelope: semantic versioning, deprecation policy, migration path

All 26 Capabilities are registered in the OctopussyCapabilityRegistry and orchestrated by the OctopussyCapabilityLifecycleManager (OCLM), which computes a topologically sorted startup order using Kahn's algorithm. Circular dependencies are detected at registration time and rejected immediately with a CircularDependencyError.
The mandatory six-phase CCA lifecycle applies to every Capability without exception:
instantiate → initialize() → inject_dependency() → start()
    → [runtime] →
stop() → cleanup()

Shutdown proceeds in strict reverse topological order.

4.0.2 OctopussyMessage
# octopussy/core/message.py
from __future__ import annotations
import hashlib
import hmac
import json
import time
from dataclasses import dataclass, field
from typing import Any, Final
from uuid import UUID, uuid4

from octopussy.core.enums import MessageType, MessagePriority

_HMAC_ALGORITHM: Final[str] = "sha256"
_MESSAGE_VERSION: Final[str] = "4.0"


@dataclass(frozen=True, slots=True)
class OctopussyMessage:
    """
    Immutable, signed inter-agent message.

    Invariants (enforced in __post_init__):
    - message_id is always a fresh UUID4 unless explicitly provided.
    - signature is always bytes (never None, never empty).
    - ttl_seconds > 0.
    - sender_id and recipient_id are valid UUIDs.
    - payload is serializable to JSON (validated on construction).
    - version is always _MESSAGE_VERSION.

    Mutations raise FrozenInstanceError. This is intentional and enforced.
    """
    message_id:    UUID
    sender_id:     UUID
    recipient_id:  UUID
    message_type:  MessageType
    payload:       dict[str, Any]           # JSON-serializable; validated in __post_init__
    trace_id:      UUID                     # propagated across the full call chain
    priority:      MessagePriority
    signature:     bytes                    # HMAC-SHA256; mandatory, never Optional
    timestamp_utc: float                    # Unix timestamp, UTC
    ttl_seconds:   int                      # message expiry; must be > 0
    version:       str = field(default=_MESSAGE_VERSION, init=False)

    def __post_init__(self) -> None:
        # Validate payload is JSON-serializable
        try:
            json.dumps(self.payload)
        except (TypeError, ValueError) as exc:
            raise ValueError(
                f"OctopussyMessage payload must be JSON-serializable: {exc}"
            ) from exc
        # Validate ttl
        if self.ttl_seconds <= 0:
            raise ValueError(
                f"ttl_seconds must be > 0, got {self.ttl_seconds}"
            )
        # Validate signature is non-empty bytes
        if not isinstance(self.signature, bytes) or len(self.signature) == 0:
            raise ValueError("signature must be non-empty bytes")

    def is_expired(self) -> bool:
        """Returns True if the message has exceeded its TTL."""
        return (time.time() - self.timestamp_utc) > self.ttl_seconds

    def verify_signature(self, secret: bytes) -> bool:
        """
        Verify HMAC-SHA256 signature.
        Uses constant-time comparison to prevent timing attacks.
        """
        expected = self._compute_signature(secret)
        return hmac.compare_digest(self.signature, expected)

    def _compute_signature(self, secret: bytes) -> bytes:
        canonical = json.dumps(
            {
                "message_id": str(self.message_id),
                "sender_id":  str(self.sender_id),
                "recipient_id": str(self.recipient_id),
                "message_type": self.message_type.value,
                "payload":    self.payload,
                "trace_id":   str(self.trace_id),
                "timestamp_utc": self.timestamp_utc,
                "ttl_seconds": self.ttl_seconds,
                "version":    self.version,
            },
            sort_keys=True,
            separators=(",", ":"),
        ).encode("utf-8")
        return hmac.new(secret, canonical, _HMAC_ALGORITHM).digest()

    @classmethod
    def create(
        cls,
        sender_id:    UUID,
        recipient_id: UUID,
        message_type: MessageType,
        payload:      dict[str, Any],
        secret:       bytes,
        trace_id:     UUID | None = None,
        priority:     MessagePriority = MessagePriority.NORMAL,
        ttl_seconds:  int = 300,
        message_id:   UUID | None = None,
    ) -> "OctopussyMessage":
        """
        Factory method. Computes HMAC signature automatically.
        This is the only correct way to create a signed OctopussyMessage.
        """
        mid       = message_id or uuid4()
        tid       = trace_id or uuid4()
        timestamp = time.time()
        # Build a temporary instance to compute signature
        # We use object.__setattr__ because the dataclass is frozen
        obj = object.__new__(cls)
        object.__setattr__(obj, "message_id",    mid)
        object.__setattr__(obj, "sender_id",     sender_id)
        object.__setattr__(obj, "recipient_id",  recipient_id)
        object.__setattr__(obj, "message_type",  message_type)
        object.__setattr__(obj, "payload",       payload)
        object.__setattr__(obj, "trace_id",      tid)
        object.__setattr__(obj, "priority",      priority)
        object.__setattr__(obj, "timestamp_utc", timestamp)
        object.__setattr__(obj, "ttl_seconds",   ttl_seconds)
        object.__setattr__(obj, "version",       _MESSAGE_VERSION)
        sig = obj._compute_signature(secret)
        object.__setattr__(obj, "signature", sig)
        obj.__post_init__()
        return obj

# ─── ObservabilityPort ────────────────────────────────────────────────────────

class ObservabilityPort(ABC):
    """
    Injected into every Capability and agent.
    All methods are fire-and-forget; they never block the caller.
    """

    @abstractmethod
    def record_metric(
        self,
        name:   str,
        value:  float,
        labels: dict[str, str] | None = None,
    ) -> None:
        """Record a metric. Non-blocking."""
        ...

    @abstractmethod
    def start_span(
        self,
        name:     str,
        trace_id: UUID,
        labels:   dict[str, str] | None = None,
    ) -> Any:  # returns a context manager
        """Start an OpenTelemetry span. Returns a context manager."""
        ...

    @abstractmethod

from __future__ import annotations
from abc import ABC, abstractmethod
from uuid import UUID

from octopussy.core.models.agent_spec import AgentHandle, AgentSpec


class AgentFactoryContract(ABC):
    """
    Provisions:
    - create_agent_from_config: full agent creation from config directory
    - validate_agent_config_dir: validation without creation (fail-fast)
    - get_agent_spec: retrieve the AgentSpec for a running agent

    Requirements:
    - ConfigurationCapability
    - LLMRouterCapability
    - MemoryCapability
    - RAGCapability
    - GraphRAGCapability
    - ToolRegistryCapability
    - SandboxCapability
    - TokenBudgetCapability
    - AgentLifecycleManagerCapability
    - SecurityCapability
    - ObservabilityCapability

    Protocols:
    - validate_agent_config_dir() MUST be called before create_agent_from_config().
    - If validation returns any errors, create_agent_from_config() raises AgentConfigurationError.
    - AgentSpec is immutable; no modification after creation.
    - agent_id is always a fresh UUID4 generated during creation.
    """

    @abstractmethod
    async def create_agent_from_config(
        self,
        config_dir: str,
    ) -> AgentHandle:
        """
        Create a fully initialized, running agent from a configuration directory.

        Process:
        1. validate_agent_config_dir() — raises AgentConfigurationError on failure
        2. Build AgentSpec via AgentSpecBuilder
        3. Resolve LLM port(s) via LLMRouterCapability
        4. Provision sandbox via SandboxCapability
        5. Register token budget via TokenBudgetCapability
        6. Initialize MemoryCapability collection for agent
        7. Initialize RAGCapability collection if rag.md present
        8. Initialize GraphRAGCapability graph if graph_rag.md present
        9. Register MCP servers if mcp_servers.json present
        10. Register agent with AgentLifecycleManagerCapability
        11. Start agent message loop (asyncio.create_task)
        12. Return AgentHandle

        Raises AgentConfigurationError with file name and line number on any error.
        """
        ...

    @abstractmethod
    async def validate_agent_config_dir(
        self,
        config_dir: str,
    ) -> list[str]:
        """
        Validate all configuration files in config_dir.
        Returns list of error strings (empty list = valid).
        Each error string includes file name and line number.
        Never raises — all errors are returned as strings.
        """
        ...

    @abstractmethod
    async def get_agent_spec(self, agent_id: UUID) -> AgentSpec:
        """
        Return the AgentSpec for a running agent.
        Raises AgentNotFoundError if agent_id is unknown.
        """
        ...


4.3 ActorCapability (OctopussyAgent)
Purpose: Defines the abstract base class for all agents and the complete agent execution model: message loop, state machine, tool execution, LLM calls, memory, RAG, MCP tool calls, and reflection.
ActorCapability
│
├── Essence

│   ├── ToolExecutor                — executes tools with permission check + sandbox

class OctopussyAgent(ABC):
    """
    Abstract base class for all Octopussy agents.

    Template Method pattern: _message_loop() defines the fixed algorithm.
    Subclasses override process_task() to implement agent-specific behavior.

    All agents:
    - Communicate exclusively via OctopussyMessage (never direct method calls)
    - Never share state with other agents
    - Never block the event loop
    - Verify every inbound message signature before processing
    - Check TTL before processing
    - Check token budget before every LLM call
    - Emit heartbeats every _HEARTBEAT_INTERVAL_SEC seconds
    - Respect LoopGuard iteration ceiling
    """

    def __init__(
        self,
        spec:              AgentSpec,
        hmac_secret:       bytes,
        inbound_queue:     asyncio.PriorityQueue,
        outbound_queue:    asyncio.Queue,
        llm_adapter:       Any,    # LLMPortAdapter
        vlm_adapter:       Any | None,  # VLMPortAdapter | None
        tool_executor:     ToolExecutorPort,
        mcp_client:        Any,    # MCPClientAdapter
        memory_store:      Any,    # MemoryStore
        rag_port:          RAGPort | None,
        lifecycle_port:    AgentLifecyclePort,
        token_budget_port: TokenBudgetPort,
        security_port:     SecurityPort,
        sandbox_port:      SandboxPort,
        observability:     ObservabilityPort,
    ) -> None:
        self._spec              = spec
        self._agent_id          = spec.agent_id
        self._hmac_secret       = hmac_secret
        self._inbound           = inbound_queue
        self._outbound          = outbound_queue
        self._llm               = llm_adapter
        self._vlm               = vlm_adapter
        self._tools             = tool_executor
        self._mcp               = mcp_client
        self._memory            = memory_store
        self._rag               = rag_port
        self._lifecycle         = lifecycle_port
        self._budget            = token_budget_port
        self._security          = security_port
        self._sandbox           = sandbox_port
        self._obs               = observability
        self._state             = AgentState.CREATED
        self._loop_guard        = LoopGuard(spec.permissions.max_loop_iterations)
        self._conversation      = ConversationMemory()
        self._shutdown_event    = asyncio.Event()
        self._pause_event       = asyncio.Event()
        self._pause_event.set()  # not paused initially
        self._task:             asyncio.Task | None = None

    # ── Public API ────────────────────────────────────────────────────────────

    async def start(self) -> None:
        """Start the agent message loop as an asyncio Task."""
        await self._lifecycle.transition_state(
            self._agent_id, AgentState.INITIALIZING
        )
        self._task = asyncio.create_task(
            self._message_loop(),
            name=f"agent-{self._agent_id}",
        )
        self._obs.log("info", f"Agent {self._spec.name} started", {
            "agent_id": str(self._agent_id),
        })

    async def stop(self, reason: str = "requested") -> None:
        """Signal the agent to stop gracefully."""
        self._shutdown_event.set()
        self._pause_event.set()  # unblock if paused
        if self._task and not self._task.done():
            try:
                await asyncio.wait_for(self._task, timeout=10.0)
            except asyncio.TimeoutError:
                self._task.cancel()
                logger.warning(
                    "Agent %s did not stop within 10s; cancelled.", self._agent_id
                )
        await self._lifecycle.transition_state(
            self._agent_id, AgentState.STOPPED, reason
        )

    async def pause(self) -> None:
        """Pause message processing."""
        self._pause_event.clear()
        await self._lifecycle.transition_state(
            self._agent_id, AgentState.PAUSED
        )

    async def resume(self) -> None:
        """Resume message processing."""
        self._pause_event.set()
        await self._lifecycle.transition_state(
            self._agent_id, AgentState.IDLE
        )

    async def enqueue(self, message: OctopussyMessage) -> None:
        """Enqueue an inbound message. Priority is derived from message.priority."""
        await self._inbound.put((message.priority.value, message))

    # ── Template Method: fixed message loop algorithm ─────────────────────────

    async def _message_loop(self) -> None:
        """
        Fixed algorithm (Template Method pattern).
        Steps:
        1. Transition to IDLE
        2. Start heartbeat emitter
        3. Loop: wait for message, apply Chain of Responsibility pipeline
        4. On shutdown: stop heartbeat, transition to STOPPING
        """
        await self._lifecycle.transition_state(self._agent_id, AgentState.IDLE)
        heartbeat_task = asyncio.create_task(self._heartbeat_loop())

        try:
            while not self._shutdown_event.is_set():
                # Wait for pause to be lifted
                await self._pause_event.wait()

                try:
                    _, message = await asyncio.wait_for(
                        self._inbound.get(), timeout=1.0
                    )
                except asyncio.TimeoutError:
                    continue

                await self._process_message(message)
                self._inbound.task_done()

        except asyncio.CancelledError:
            pass
        finally:
            heartbeat_task.cancel()
            try:
                await heartbeat_task
            except asyncio.CancelledError:
                pass
            await self._lifecycle.transition_state(
                self._agent_id, AgentState.STOPPING
            )

    async def _process_message(self, message: OctopussyMessage) -> None:
        """
        Chain of Responsibility pipeline:
        1. Signature verification
        2. TTL check
        3. Control message handling
        4. Prompt injection scan
        5. Budget check
        6. Task processing
        """
        with self._obs.start_span("process_message", message.trace_id, {
            "agent_id": str(self._agent_id),
            "message_type": message.message_type.value,
        }):
            # Step 1: Signature verification
            if not message.verify_signature(self._hmac_secret):
                self._obs.log("warning", "Message signature verification failed", {
                    "agent_id": str(self._agent_id),
                    "message_id": str(message.message_id),
                })
                self._obs.audit(
                    "MESSAGE_SIGNATURE_FAILURE",
                    message.sender_id,
                    str(self._agent_id),
                    "DENIED",
                    {"message_id": str(message.message_id)},
                )
                return  # silently drop; already logged and audited

            # Step 2: TTL check
            if message.is_expired():
                self._obs.log("warning", "Message TTL expired", {
                    "agent_id": str(self._agent_id),
                    "message_id": str(message.message_id),
                })
                return

            # Step 3: Control message handling
            if message.message_type == MessageType.CONTROL:
                await self._handle_control(message)
                return

            # Step 4: Prompt injection scan
            prompt_text = message.payload.get("content", "")
            is_safe, reason = await self._security.scan_prompt(prompt_text)
            if not is_safe:
                self._obs.log("warning", "Prompt injection detected", {
                    "agent_id": str(self._agent_id),
                    "reason": reason,
                })
                self._obs.audit(
                    "PROMPT_INJECTION_DETECTED",
                    message.sender_id,
                    str(self._agent_id),
                    "BLOCKED",
                    {"reason": reason},
                )
                await self._send_error(message, f"Prompt rejected: {reason}")
                return

            # Step 5: Budget check
            estimated_tokens = len(prompt_text.split()) * 2  # rough estimate
            if not await self._budget.check_budget(self._agent_id, estimated_tokens):
                self._obs.log("warning", "Token budget exhausted", {
                    "agent_id": str(self._agent_id),
                })
                await self._send_error(message, "Token budget exhausted")
                return

            # Step 6: Task processing
            await self._lifecycle.transition_state(
                self._agent_id, AgentState.PROCESSING
            )
            try:
                self._loop_guard.reset()
                result = await self.process_task(message)
                await self._send_result(message, result)
            except LoopLimitExceededError as exc:
                self._obs.log("error", str(exc), {"agent_id": str(self._agent_id)})
                await self._send_error(message, str(exc))
            except Exception as exc:
                self._obs.log("error", f"Task processing failed: {exc}", {
                    "agent_id": str(self._agent_id),
                    "exc_type": type(exc).__name__,
                })
                await self._send_error(message, str(exc))
            finally:
                await self._lifecycle.transition_state(
                    self._agent_id, AgentState.IDLE
                )

    async def _handle_control(self, message: OctopussyMessage) -> None:
        """Handle CONTROL messages (PAUSE, RESUME, STOP, STATUS)."""
        command = message.payload.get("command", "")
        match command:
            case ControlCommand.PAUSE:
                await self.pause()
            case ControlCommand.RESUME:
                await self.resume()
            case ControlCommand.STOP:
                await self.stop("control message")
            case ControlCommand.STATUS:
                info = await self._lifecycle.get_agent_info(self._agent_id)
                await self._send_result(message, {"status": info})
            case _:
                self._obs.log("warning", f"Unknown control command: {command!r}")

    async def _heartbeat_loop(self) -> None:
        """Emit heartbeat messages at regular intervals."""
        while not self._shutdown_event.is_set():
            await asyncio.sleep(_HEARTBEAT_INTERVAL_SEC)
            self._obs.record_metric(
                "agent.heartbeat",
                1.0,
                {"agent_id": str(self._agent_id), "name": self._spec.name},
            )

    async def _send_result(
        self,
        original: OctopussyMessage,
        result:   Any,
    ) -> None:
        """Send a RESULT message back to the original sender."""
        response = OctopussyMessage.create(
            sender_id    = self._agent_id,
            recipient_id = original.sender_id,
            message_type = MessageType.RESULT,
            payload      = {"result": result, "task_id": str(original.message_id)},
            secret       = self._hmac_secret,
            trace_id     = original.trace_id,
            priority     = original.priority,
        )
        await self._outbound.put(response)

    async def _send_error(
        self,
        original: OctopussyMessage,
        error:    str,
    ) -> None:
        """Send an ERROR message back to the original sender."""
        response = OctopussyMessage.create(
            sender_id    = self._agent_id,
            recipient_id = original.sender_id,
            message_type = MessageType.ERROR,
            payload      = {"error": error, "task_id": str(original.message_id)},
            secret       = self._hmac_secret,
            trace_id     = original.trace_id,
            priority     = MessagePriority.HIGH,
        )
        await self._outbound.put(response)

    # ── Abstract methods (subclasses implement) ───────────────────────────────

    @abstractmethod
    async def process_task(self, message: OctopussyMessage) -> Any:
        """
        Process a task message and return a result.
        Called by _message_loop after all pipeline checks pass.
        Subclasses implement agent-specific behavior here.
        May call self._call_llm(), self._call_tool(), self._retrieve_rag(),
        self._call_mcp_tool() as needed.
        Must respect self._loop_guard.
        """
        ...

    # ── Protected helpers for subclasses ─────────────────────────────────────

    async def _call_llm(
        self,
        system_prompt: str,
        user_prompt:   str,
        trace_id:      UUID,
        tools:         list[dict] | None = None,
    ) -> LLMCompletion:
        """
        Call the LLM via LLMPortAdapter (retry + circuit breaker).
        Records token usage via TokenBudgetPort.
        Raises AgentBudgetExhaustedError if budget check fails.
        """
        self._loop_guard.increment()
        request = LLMRequest(
            system_prompt  = system_prompt,
            user_prompt    = user_prompt,
            conversation   = tuple(self._conversation.get_history()),
            max_tokens     = self._spec.intelligence.primary.max_tokens,
            temperature    = self._spec.intelligence.primary.temperature,
            trace_id       = trace_id,
            tools          = tuple(tools or []),
        )
        completion = await self._llm.complete(request)
        await self._budget.record_usage(
            self._agent_id,
            completion.input_tokens,
            completion.output_tokens,
            completion.cost_usd,
        )
        self._conversation.add(
            {"role": "user",      "content": user_prompt},
            {"role": "assistant", "content": completion.content},
        )
        return completion

    async def _call_tool(
        self,
        tool_id:    str,
        parameters: dict[str, Any],
        context:    AgentContext,
    ) -> Any:
        """Execute a registered tool via ToolExecutorPort."""
        return await self._tools.execute(tool_id, parameters, context)

    async def _call_mcp_tool(
        self,

    ) -> Any:

        return await self._mcp.call_tool(server_id, tool_name, arguments, trace_id)

    async def _retrieve_rag(
        self,
        query: str,
        top_k: int = 5,
    ) -> list[Any]:
        """Retrieve relevant documents from the agent's RAG collection."""
        if self._rag is None or self._spec.rag is None:
            return []
        return await self._rag.retrieve(
            self._agent_id,
            query,
            self._spec.rag.collection_name,
            top_k,
        )


# ─── LoopGuard ────────────────────────────────────────────────────────────────

class LoopGuard:
    """
    Prevents infinite loops by enforcing an iteration ceiling.
    Raises LoopLimitExceededError when the ceiling is reached.
    """

    def __init__(self, max_iterations: int) -> None:
        if max_iterations <= 0:
            raise ValueError(f"max_iterations must be > 0, got {max_iterations}")
        self._max  = max_iterations
        self._count = 0

    def increment(self) -> None:
        self._count += 1
        if self._count > self._max:
            raise LoopLimitExceededError(UUID(int=0), self._max)  # agent_id injected by caller

    def reset(self) -> None:
        self._count = 0

    @property
    def iteration(self) -> int:
        return self._count


# ─── ConversationMemory ───────────────────────────────────────────────────────

class ConversationMemory:
    """
    Short-term sliding-window conversation history.
    Thread-safe for asyncio (single-threaded event loop).
    """

    def __init__(self, max_turns: int = 20) -> None:
        self._max   = max_turns
        self._turns: list[dict[str, str]] = []

    def add(self, *turns: dict[str, str]) -> None:
        for turn in turns:
            self._turns.append(turn)
        # Trim to max_turns pairs (each pair = user + assistant)
        if len(self._turns) > self._max * 2:
            self._turns = self._turns[-(self._max * 2):]

    def get_history(self) -> list[dict[str, str]]:
        return list(self._turns)

    def clear(self) -> None:
        self._turns.clear()

4.3.1 Agent State Machine
# octopussy/capabilities/actor/essence/state_machine.py
from __future__ import annotations
from uuid import UUID

from octopussy.core.enums import AgentState
from octopussy.core.exceptions.agent_exceptions import InvalidStateTransitionError

# Valid state transitions: from_state → set of valid to_states
_VALID_TRANSITIONS: dict[AgentState, frozenset[AgentState]] = {
    AgentState.CREATED:      frozenset({AgentState.INITIALIZING}),
    AgentState.INITIALIZING: frozenset({AgentState.IDLE, AgentState.ERROR}),
    AgentState.IDLE:         frozenset({AgentState.PROCESSING, AgentState.PAUSED,
                                        AgentState.SLEEPING, AgentState.STOPPING,
                                        AgentState.ERROR}),
    AgentState.PROCESSING:   frozenset({AgentState.IDLE, AgentState.STOPPING,
                                        AgentState.ERROR}),
    AgentState.PAUSED:       frozenset({AgentState.IDLE, AgentState.STOPPING}),
    AgentState.SLEEPING:     frozenset({AgentState.IDLE, AgentState.STOPPING}),
    AgentState.STOPPING:     frozenset({AgentState.STOPPED}),
    AgentState.STOPPED:      frozenset(),   # terminal state
    AgentState.ERROR:        frozenset({AgentState.STOPPING, AgentState.IDLE}),
}


class AgentStateMachine:
    """
    Strict, validated agent state machine.
    All transitions are validated against _VALID_TRANSITIONS.
    Invalid transitions raise InvalidStateTransitionError immediately.
    """

    def __init__(self, agent_id: UUID, initial: AgentState = AgentState.CREATED) -> None:
        self._agent_id = agent_id
        self._state    = initial

    @property
    def state(self) -> AgentState:
        return self._state

    def transition(self, new_state: AgentState, reason: str = "") -> None:
        """
        Transition to new_state.
        Raises InvalidStateTransitionError if transition is not in _VALID_TRANSITIONS.
        """
        allowed = _VALID_TRANSITIONS.get(self._state, frozenset())
        if new_state not in allowed:
            raise InvalidStateTransitionError(
                self._agent_id, self._state.value, new_state.value
            )
        self._state = new_state

    def can_transition(self, new_state: AgentState) -> bool:
        """Returns True if transition to new_state is valid from current state."""
        return new_state in _VALID_TRANSITIONS.get(self._state, frozenset())

4.3.2 LLMPortAdapter (Circuit Breaker + Retry)
# octopussy/capabilities/actor/realization/llm_port_adapter.py
from __future__ import annotations

import asyncio
import logging
import time
from typing import Final

from octopussy.core.enums import CircuitBreakerState
from octopussy.core.exceptions.octopussy_exceptions import LLMProviderError, LLMTimeoutError
from octopussy.core.ports.llm_ports import LLMCompletion, LLMPort, LLMRequest

logger = logging.getLogger(__name__)

_MAX_RETRIES:        Final[int]   = 3
_BASE_BACKOFF_SEC:   Final[float] = 1.0
_FAILURE_THRESHOLD:  Final[int]   = 5
_RECOVERY_TIMEOUT:   Final[float] = 60.0


class LLMPortAdapter:
    """
    Wraps an LLMPort with:
    - Exponential backoff retry (up to _MAX_RETRIES attempts)
    - Circuit breaker (CLOSED → OPEN after _FAILURE_THRESHOLD failures)
    - Automatic fallback to secondary LLMPort when primary circuit is OPEN
    - Full observability via ObservabilityPort
    """

    def __init__(
        self,
        primary:       LLMPort,
        fallback:      LLMPort | None,
        observability: Any,  # ObservabilityPort
    ) -> None:
        self._primary      = primary
        self._fallback     = fallback
        self._obs          = observability
        self._cb_state     = CircuitBreakerState.CLOSED
        self._failure_count = 0
        self._last_failure  = 0.0

    async def complete(self, request: LLMRequest) -> LLMCompletion:
        """
        Complete with circuit breaker + retry + fallback.
        Raises LLMProviderError if all attempts and fallback fail.
        """
        if self._cb_state == CircuitBreakerState.OPEN:
            if time.time() - self._last_failure > _RECOVERY_TIMEOUT:
                self._cb_state = CircuitBreakerState.HALF_OPEN
            elif self._fallback:
                return await self._fallback.complete(request)
            else:
                raise LLMProviderError(
                    f"Circuit breaker OPEN for {self._primary.provider_name}"
                )

        last_exc: Exception | None = None
        for attempt in range(1, _MAX_RETRIES + 1):
            try:
                result = await self._primary.complete(request)
                self._on_success()
                return result
            except (LLMProviderError, LLMTimeoutError) as exc:
                last_exc = exc
                self._on_failure()
                if attempt < _MAX_RETRIES:
                    backoff = _BASE_BACKOFF_SEC * (2 ** (attempt - 1))
                    logger.warning(
                        "LLM call failed (attempt %d/%d), retrying in %.1fs: %s",
                        attempt, _MAX_RETRIES, backoff, exc,
                    )
                    await asyncio.sleep(backoff)

        # All retries exhausted; try fallback
        if self._fallback:
            logger.warning("Primary LLM exhausted retries; using fallback.")
            return await self._fallback.complete(request)

        raise LLMProviderError(
            f"All {_MAX_RETRIES} attempts failed for {self._primary.provider_name}"
        ) from last_exc

    def _on_success(self) -> None:
        self._failure_count = 0
        self._cb_state      = CircuitBreakerState.CLOSED

    def _on_failure(self) -> None:
        self._failure_count += 1
        self._last_failure   = time.time()
        if self._failure_count >= _FAILURE_THRESHOLD:
            self._cb_state = CircuitBreakerState.OPEN
            logger.error(
                "Circuit breaker OPEN for %s after %d failures.",
                self._primary.provider_name, self._failure_count,
            )

Security invariants:

Every OctopussyMessage carries a mandatory HMAC-SHA256 signature (bytes, never Optional[bytes]).
Every message is verified before processing; failed verification is logged, audited, and the message is silently dropped.
TOTP is mandatory for SUPERUSER and ADMIN roles.
Every prompt is scanned for injection before reaching an LLM.
Every agent output is validated by the guardrail engine before delivery.
All secrets are stored in SecretsCapability; never in configuration files or environment variables.
mTLS is enforced on all network endpoints (ports 47200, 47201, 47202).

5. ACTOR MODEL &amp; AGENT ARCHITECTURE
Every Octopussy agent is an Actor: a long-lived, autonomous entity that:

Communicates exclusively via OctopussyMessage (never direct method calls)
Owns its state exclusively (no shared mutable state with other agents)
Processes messages asynchronously from its PriorityInboundQueue
Never blocks the asyncio event loop
Has its own identity (agent_id), memory, permissions, budget, and lifecycle

Three agent types:



Type
Description
Execution Mode



STATEFUL
Maintains state across tasks; long-lived
LOOP


STATELESS
No state between tasks; each task is independent
ONE_SHOT or LOOP


DAEMON
Runs indefinitely; typically scheduler-triggered
LOOP


Concrete agent implementations (all derive from OctopussyAgent):
# octopussy/capabilities/actor/essence/agent_types.py
from __future__ import annotations
from typing import Any

from octopussy.core.message import OctopussyMessage
from octopussy.capabilities.actor.essence.octopussy_agent import OctopussyAgent


class StatelessOneShot(OctopussyAgent):
    """
    Executes a single task and terminates.
    Used for: research tasks, code generation, one-off analyses.
    """
    async def process_task(self, message: OctopussyMessage) -> Any:
        task = message.payload.get("content", "")
        docs = await self._retrieve_rag(task)
        rag_context = "\n".join(d.content for d in docs) if docs else ""
        completion = await self._call_llm(
            system_prompt = self._spec.goal,
            user_prompt   = f"Context:\n{rag_context}\n\nTask:\n{task}" if rag_context else task,
            trace_id      = message.trace_id,
        )
        return completion.content


class StatefulLoop(OctopussyAgent):
    """
    Maintains state across tasks. Runs until explicitly stopped.
    Used for: conversational agents, monitoring agents.
    """
    async def process_task(self, message: OctopussyMessage) -> Any:
        task = message.payload.get("content", "")
        # Retrieve relevant memory
        memory_context = await self._retrieve_rag(task)
        context_str = "\n".join(d.content for d in memory_context) if memory_context else ""
        completion = await self._call_llm(
            system_prompt = self._spec.goal,
            user_prompt   = task,
            trace_id      = message.trace_id,
        )
        # Persist result to long-term memory
        await self._memory.store(
            self._agent_id,
            f"task_{message.message_id}",
            {"task": task, "result": completion.content},
        )
        return completion.content


class ScheduledTaskDaemon(OctopussyAgent):
    """
    Daemon agent triggered by SchedulerCapability.
    Used for: morning briefings, periodic reports, monitoring.
    """
    async def process_task(self, message: OctopussyMessage) -> Any:
        task = message.payload.get("content", "")
        # Execute scheduled task with tool calls
        completion = await self._call_llm(
            system_prompt = self._spec.goal,
            user_prompt   = task,
            trace_id      = message.trace_id,
        )
        return completion.content

10.2 HMAC Signing
# All message signing uses OctopussyMessage.create() factory method.
# The HMAC key is loaded from SecretsCapability at startup.
# Key rotation is supported via SecretsCapability.rotate_secret().
# Signature algorithm: HMAC-SHA256 over canonical JSON representation.

10.3 Sandbox Isolation (Bulkhead Pattern)

11. MCP INTEGRATION ARCHITECTURE (2025-11-25)
Octopussy implements the MCP 2025-11-25 specification (Streamable HTTP transport) for both server-side (exposing Octopussy tools to external MCP clients) and client-side (calling external MCP tool servers from agents).
11.1 MCP Client (Agent → External MCP Server)
Agents call external MCP servers via MCPClientAdapter. All calls use the MCP 2025-11-25 Streamable HTTP transport. LLM-dependent tool calling mechanisms are not used — all tool calls go through MCP.
# octopussy/capabilities/actor/realization/mcp_client_adapter.py
from __future__ import annotations

import asyncio
import json
import logging
from typing import Any
from uuid import UUID, uuid4

import httpx

from octopussy.core.exceptions.octopussy_exceptions import (
    MCPConnectionError,
    MCPToolCallError,
)
from octopussy.core.models.agent_spec import MCPServerSpec

logger = logging.getLogger(__name__)

_MCP_PROTOCOL_VERSION = "2025-11-25"


class MCPClientAdapter:
    """
    MCP 2025-11-25 Streamable HTTP client.

    Protocol details:
    - Single HTTP endpoint supporting both POST and GET.
    - POST for tool calls (tools/call) and tool discovery (tools/list).
    - GET for SSE streaming (server-to-client notifications).
    - MCP-Protocol-Version header: "2025-11-25" on all requests.
    - MCP-Session-Id header: assigned by server during initialize, sent on all subsequent requests.
    - JSON-RPC 2.0 message encoding (UTF-8).
    - Session initialized on first connect; session_id persisted per server.

    All tool calls go through MCP. LLM-dependent tool calling is not used.
    """

    def __init__(
        self,
        server_specs:  list[MCPServerSpec],
        secrets_port:  Any,   # SecretsPort
        observability: Any,   # ObservabilityPort
    ) -> None:
        self._specs        = {s.server_id: s for s in server_specs}
        self._secrets      = secrets_port
        self._obs          = observability
        self._sessions:    dict[str, str]         = {}   # server_id → session_id
        self._tool_cache:  dict[str, list[dict]]  = {}   # server_id → tools list
        self._http_clients: dict[str, httpx.AsyncClient] = {}

    async def initialize(self) -> None:
        """Initialize MCP sessions for all configured servers."""
        for server_id, spec in self._specs.items():
            await self._initialize_session(server_id, spec)

    async def _initialize_session(
        self,
        server_id: str,
        spec:      MCPServerSpec,
    ) -> None:
        """
        Initialize MCP session per 2025-11-25 spec.
        Sends initialize request, receives session_id from MCP-Session-Id header.
        """
        headers = {
            "Content-Type":        "application/json",
            "MCP-Protocol-Version": _MCP_PROTOCOL_VERSION,
        }
        if spec.api_key_secret:
            api_key = await self._secrets.get_secret(spec.api_key_secret)
            headers["Authorization"] = f"Bearer {api_key}"

        client = httpx.AsyncClient(
            timeout=spec.timeout_seconds,
            headers=headers,
        )
        self._http_clients[server_id] = client

        init_payload = {
            "jsonrpc": "2.0",
            "id":      str(uuid4()),
            "method":  "initialize",
            "params": {
                "protocolVersion": _MCP_PROTOCOL_VERSION,
                "capabilities": {
                    "tools": {},
                },
                "clientInfo": {
                    "name":    "octopussy",
                    "version": "4.0.0",
                },
            },
        }

        try:
            response = await client.post(spec.endpoint_url, json=init_payload)
            response.raise_for_status()

            # Extract session ID from response header (2025-11-25 spec)
            session_id = response.headers.get("MCP-Session-Id")
            if session_id:
                self._sessions[server_id] = session_id
                logger.info(
                    "MCP session initialized: server=%s session=%s",
                    server_id, session_id,
                )

            # Cache tool list
            await self._refresh_tool_list(server_id, spec)

        except httpx.HTTPError as exc:
            raise MCPConnectionError(
                f"Failed to initialize MCP session for server {server_id!r}: {exc}"
            ) from exc

    async def _refresh_tool_list(
        self,
        server_id: str,
        spec:      MCPServerSpec,
    ) -> None:
        """Discover available tools from MCP server (tools/list)."""
        response = await self._post(
            server_id,
            spec,
            method="tools/list",
            params={},
        )
        tools = response.get("result", {}).get("tools", [])
        # Apply tool name prefix if configured
        if spec.tool_name_prefix:
            for tool in tools:
                tool["name"] = f"{spec.tool_name_prefix}{tool['name']}"
        self._tool_cache[server_id] = tools
        logger.info(
            "MCP tools discovered: server=%s count=%d",
            server_id, len(tools),
        )

    async def call_tool(
        self,
        server_id:  str,
        tool_name:  str,
        arguments:  dict[str, Any],
        trace_id:   UUID,
    ) -> Any:
        """
        Call a tool on an MCP server (tools/call, 2025-11-25).
        Returns the tool result content.
        Raises MCPToolCallError on failure.
        """
        if server_id not in self._specs:
            raise MCPToolCallError(
                f"Unknown MCP server: {server_id!r}"
            )
        spec = self._specs[server_id]

        with self._obs.start_span("mcp.tool_call", trace_id, {
            "server_id": server_id,
            "tool_name": tool_name,
        }):
            try:
                response = await self._post(
                    server_id,
                    spec,
                    method="tools/call",
                    params={
                        "name":      tool_name,
                        "arguments": arguments,
                    },
                )
                result = response.get("result", {})
                if result.get("isError"):
                    raise MCPToolCallError(
                        f"MCP tool {tool_name!r} on {server_id!r} returned error: "
                        f"{result.get('content', 'unknown error')}"
                    )
                self._obs.record_metric(
                    "mcp.tool_call.success", 1.0,
                    {"server_id": server_id, "tool_name": tool_name},
                )
                return result.get("content")

            except MCPToolCallError:
                raise
            except Exception as exc:
                self._obs.record_metric(
                    "mcp.tool_call.error", 1.0,
                    {"server_id": server_id, "tool_name": tool_name},
                )
                raise MCPToolCallError(
                    f"MCP tool call failed: server={server_id!r} tool={tool_name!r}: {exc}"
                ) from exc

    async def list_tools(self, server_id: str) -> list[dict]:
        """Return cached tool list for a server."""
        return self._tool_cache.get(server_id, [])

    async def list_all_tools(self) -> dict[str, list[dict]]:
        """Return all tools from all configured MCP servers."""
        return dict(self._tool_cache)

    async def _post(
        self,
        server_id: str,
        spec:      MCPServerSpec,
        method:    str,
        params:    dict[str, Any],
    ) -> dict[str, Any]:
        """
        Send a JSON-RPC 2.0 POST request to the MCP server.
        Includes MCP-Protocol-Version and MCP-Session-Id headers.
        """
        client = self._http_clients[server_id]
        headers: dict[str, str] = {
            "MCP-Protocol-Version": _MCP_PROTOCOL_VERSION,
        }
        if server_id in self._sessions:
            headers["MCP-Session-Id"] = self._sessions[server_id]

        payload = {
            "jsonrpc": "2.0",
            "id":      str(uuid4()),
            "method":  method,
            "params":  params,
        }

        try:
            response = await client.post(
                spec.endpoint_url,
                json=payload,
                headers=headers,
            )
            response.raise_for_status()
            return response.json()
        except httpx.HTTPStatusError as exc:
            raise MCPToolCallError(
                f"MCP HTTP error {exc.response.status_code} for {method!r}: {exc}"
            ) from exc
        except httpx.HTTPError as exc:
            raise MCPConnectionError(
                f"MCP connection error for {method!r} on {server_id!r}: {exc}"
            ) from exc

    async def close(self) -> None:
        """Close all HTTP client sessions."""
        for client in self._http_clients.values():
            await client.aclose()
        self._http_clients.clear()
        self._sessions.clear()

11.2 MCP Server (Octopussy → External MCP Clients)
Octopussy exposes its tool registry as an MCP server, allowing external MCP clients (other AI systems, IDEs, etc.) to discover and call Octopussy tools.
# octopussy/capabilities/plugin/mcp_server.py
from __future__ import annotations

import json
import logging
from typing import Any
from uuid import uuid4

from fastapi import FastAPI, Request, Response
from fastapi.responses import StreamingResponse

from octopussy.core.exceptions.octopussy_exceptions import MCPError

logger = logging.getLogger(__name__)

_MCP_PROTOCOL_VERSION = "2025-11-25"


class OctopussyMCPServer:
    """
    Exposes Octopussy's ToolRegistryCapability as an MCP 2025-11-25 server.
    Implements Streamable HTTP transport.
    Single endpoint: /mcp (POST + GET)
    """

    def __init__(
        self,
        tool_registry: Any,   # ToolRegistryContract
        security_port: Any,   # SecurityPort
        observability: Any,   # ObservabilityPort
        port:          int = 47203,
    ) -> None:
        self._tools    = tool_registry
        self._security = security_port
        self._obs      = observability
        self._port     = port
        self._sessions: dict[str, dict] = {}   # session_id → session metadata
        self._app      = FastAPI(title="Octopussy MCP Server")
        self._register_routes()

    def _register_routes(self) -> None:
        """Register MCP endpoint routes."""

        @self._app.post("/mcp")
        async def mcp_post(request: Request) -> Response:
            return await self._handle_post(request)

        @self._app.get("/mcp")
        async def mcp_get(request: Request) -> StreamingResponse:
            return await self._handle_get(request)

    async def _handle_post(self, request: Request) -> Response:
        """
        Handle MCP POST requests (initialize, tools/list, tools/call).
        Returns JSON-RPC 2.0 response.
        """
        protocol_version = request.headers.get("MCP-Protocol-Version", "")
        if protocol_version != _MCP_PROTOCOL_VERSION:
            return Response(
                content=json.dumps({
                    "jsonrpc": "2.0",
                    "error": {
                        "code": -32600,
                        "message": f"Unsupported protocol version: {protocol_version!r}. "
                                   f"Expected: {_MCP_PROTOCOL_VERSION!r}",
                    },
                    "id": None,
                }),
                media_type="application/json",
                status_code=400,
            )

        body = await request.json()
        method  = body.get("method", "")
        params  = body.get("params", {})
        rpc_id  = body.get("id")

        match method:
            case "initialize":
                return await self._handle_initialize(rpc_id, params)
            case "tools/list":
                return await self._handle_tools_list(request, rpc_id)
            case "tools/call":
                return await self._handle_tools_call(request, rpc_id, params)
            case _:
                return Response(
                    content=json.dumps({
                        "jsonrpc": "2.0",
                        "error": {"code": -32601, "message": f"Method not found: {method!r}"},
                        "id": rpc_id,
                    }),
                    media_type="application/json",
                    status_code=404,
                )

    async def _handle_initialize(self, rpc_id: Any, params: dict) -> Response:
        """Handle MCP initialize request. Assigns session ID."""
        session_id = str(uuid4())
        self._sessions[session_id] = {
            "client_info": params.get("clientInfo", {}),
            "protocol_version": params.get("protocolVersion", _MCP_PROTOCOL_VERSION),
        }
        response_body = json.dumps({
            "jsonrpc": "2.0",
            "id":      rpc_id,
            "result": {
                "protocolVersion": _MCP_PROTOCOL_VERSION,
                "capabilities": {
                    "tools": {"listChanged": True},
                },
                "serverInfo": {
                    "name":    "octopussy",
                    "version": "4.0.0",
                },
            },
        })
        return Response(
            content=response_body,
            media_type="application/json",
            headers={"MCP-Session-Id": session_id},
        )

    async def _handle_tools_list(self, request: Request, rpc_id: Any) -> Response:
        """Handle tools/list request. Returns all registered tools."""
        tools = await self._tools.list_tools()
        mcp_tools = [
            {
                "name":        t.tool_id,
                "description": t.description,
                "inputSchema": t.parameters,
            }
            for t in tools
        ]
        return Response(
            content=json.dumps({
                "jsonrpc": "2.0",
                "id":      rpc_id,
                "result":  {"tools": mcp_tools},
            }),
            media_type="application/json",
        )

    async def _handle_tools_call(
        self,
        request: Request,
        rpc_id:  Any,
        params:  dict,
    ) -> Response:
        """Handle tools/call request. Executes tool and returns result."""
        tool_name = params.get("name", "")
        arguments = params.get("arguments", {})

        try:
            # MCP tool calls use a system agent context
            from octopussy.capabilities.actor.essence.octopussy_agent import AgentContext
            from uuid import uuid4 as _uuid4
            ctx = AgentContext(
                agent_id   = _uuid4(),  # system context
                session_id = _uuid4(),
                trace_id   = _uuid4(),
                spec       = None,      # type: ignore[arg-type]
            )
            result = await self._tools.execute_tool(tool_name, arguments, ctx)
            return Response(
                content=json.dumps({
                    "jsonrpc": "2.0",
                    "id":      rpc_id,
                    "result": {
                        "content": [{"type": "text", "text": str(result.output)}],
                        "isError": not result.success,
                    },
                }),
                media_type="application/json",
            )
        except Exception as exc:
            return Response(
                content=json.dumps({
                    "jsonrpc": "2.0",
                    "id":      rpc_id,
                    "result": {
                        "content": [{"type": "text", "text": str(exc)}],
                        "isError": True,
                    },
                }),
                media_type="application/json",
            )

    async def _handle_get(self, request: Request) -> StreamingResponse:
        """
        Handle MCP GET requests for SSE streaming (server-to-client notifications).
        Used for tool list change notifications.
        """
        async def event_stream():
            yield "data: {\"type\": \"connected\"}\n\n"
            # In production: yield tool list change events as tools are hot-loaded
            while True:
                import asyncio
                await asyncio.sleep(30)
                yield ": keepalive\n\n"

        return StreamingResponse(
            event_stream(),
            media_type="text/event-stream",
            headers={
                "Cache-Control": "no-cache",
                "X-Accel-Buffering": "no",
            },
        )

# octopussy/plugins/comm_adapters/slack/slack_adapter.py
from __future__ import annotations
import asyncio
import logging
from typing import Any
from uuid import uuid4

from slack_bolt.async_app import AsyncApp
from slack_bolt.adapter.fastapi.async_handler import AsyncSlackRequestHandler

from octopussy.capabilities.comm_adapter.essence.base_adapter import CommunicationAdapter
from octopussy.core.enums import MessagePriority, MessageType
from octopussy.core.message import OctopussyMessage

logger = logging.getLogger(__name__)


class SlackAdapter(CommunicationAdapter):
    """
    Bidirectional Slack communication adapter.
    Implements COMM_ADAPTER_PLUGIN.
    """

    def __init__(
        self,
        config:        dict[str, Any],
        secrets_port:  Any,    # SecretsPort
        orchestrator:  Any,    # OrchestratorContract
        hmac_secret:   bytes,
        system_id:     Any,    # UUID
        observability: Any,    # ObservabilityPort
    ) -> None:
        self._config       = config
        self._secrets      = secrets_port
        self._orchestrator = orchestrator
        self._hmac_secret  = hmac_secret
        self._system_id    = system_id
        self._obs          = observability
        self._app:         AsyncApp | None = None
        self._handler:     AsyncSlackRequestHandler | None = None

    @property
    def channel_id(self) -> str:
        return "slack"

    async def start(self) -> None:
        """Initialize Slack Bolt app and start listening."""
        bot_token     = await self._secrets.get_secret(
            self._config["bot_token_secret"]
        )
        signing_secret = await self._secrets.get_secret(
            self._config["signing_secret_name"]
        )
        self._app = AsyncApp(token=bot_token, signing_secret=signing_secret)
        self._handler = AsyncSlackRequestHandler(self._app)

        @self._app.event("app_mention")
        async def handle_mention(event: dict, say: Any) -> None:
            await self._on_slack_message(event, say)

        @self._app.event("message")
        async def handle_dm(event: dict, say: Any) -> None:
            if event.get("channel_type") == "im":
                await self._on_slack_message(event, say)

        logger.info("Slack adapter started.")

    async def stop(self) -> None:
        logger.info("Slack adapter stopped.")

    async def send_message(
        self,
        external_user_id: str,
        content:          str,
        metadata:         dict | None = None,
    ) -> None:
        """Send a message to a Slack channel or user."""
        if self._app:
            channel = metadata.get("channel") if metadata else external_user_id
            await self._app.client.chat_postMessage(
                channel=channel,
                text=content,
            )

    async def on_inbound(self, message: OctopussyMessage) -> None:
        """Receive a response from Octopussy and send to Slack."""
        result  = message.payload.get("result", "")
        channel = message.payload.get("slack_channel", "")
        if channel and self._app:
            await self._app.client.chat_postMessage(
                channel=channel,
                text=str(result),
            )

    async def _on_slack_message(self, event: dict, say: Any) -> None:
        """Translate Slack message to OctopussyMessage and submit to orchestrator."""
        text       = event.get("text", "").strip()
        user_id    = event.get("user", "")
        channel    = event.get("channel", "")
        trace_id   = uuid4()

        # Find default agent
        default_agent = self._config.get("default_agent", "")

        try:
            task_handle = await self._orchestrator.submit_task(
                task_description = text,
                trace_id         = trace_id,
                caller_token     = "",  # internal adapter call
            )
            # Response will be delivered via on_inbound() callback
            self._obs.log("info", "Slack message submitted", {
                "user_id":  user_id,
                "task_id":  str(task_handle.task_id),
                "trace_id": str(trace_id),
            })
        except Exception as exc:
            logger.error("Failed to submit Slack message: %s", exc)
            await say(f"Sorry, I encountered an error: {exc}")

# octopussy/core/registry.py
from __future__ import annotations

import logging
from collections import defaultdict, deque
from typing import Any, Type

from octopussy.core.exceptions.octopussy_exceptions import (
    CapabilityNotFoundError,
    CircularDependencyError,
)

logger = logging.getLogger(__name__)


class CapabilityRegistration:
    """Holds metadata for a registered Capability."""
    __slots__ = ("name", "instance", "dependencies", "version")

    def __init__(
        self,
        name:         str,
        instance:     Any,
        dependencies: list[str],
        version:      str,
    ) -> None:
        self.name         = name
        self.instance     = instance
        self.dependencies = dependencies
        self.version      = version


class OctopussyCapabilityRegistry:
    """
    Registry for all Octopussy Capabilities.
    Detects circular dependencies at registration time (Kahn's algorithm).
    Thread-safe for asyncio (single-threaded event loop).
    """

    def __init__(self) -> None:
        self._capabilities: dict[str, CapabilityRegistration] = {}

    def register(
        self,
        name:         str,
        instance:     Any,
        dependencies: list[str],
        version:      str = "4.0",
    ) -> None:
        """
        Register a Capability.
        Raises CircularDependencyError if registration creates a cycle.
        """
        if name in self._capabilities:
            raise ValueError(f"Capability {name!r} is already registered.")
        self._capabilities[name] = CapabilityRegistration(
            name, instance, dependencies, version
        )
        # Validate no cycles after registration
        try:
            self.topological_order()
        except CircularDependencyError:
            del self._capabilities[name]
            raise

    def get(self, name: str) -> Any:
        """
        Retrieve a Capability instance by name.
        Raises CapabilityNotFoundError if not registered.
        """
        if name not in self._capabilities:
            raise CapabilityNotFoundError(
                f"Capability {name!r} is not registered."
            )
        return self._capabilities[name].instance

    def topological_order(self) -> list[str]:
        """
        Compute topological startup order using Kahn's algorithm.
        Raises CircularDependencyError if a cycle is detected.
        Returns list of Capability names in startup order.
        """
        in_degree: dict[str, int] = {name: 0 for name in self._capabilities}
        adjacency: dict[str, list[str]] = defaultdict(list)

        for name, reg in self._capabilities.items():
            for dep in reg.dependencies:
                if dep not in self._capabilities:
                    raise CapabilityNotFoundError(
                        f"Capability {name!r} depends on {dep!r}, which is not registered."
                    )
                adjacency[dep].append(name)
                in_degree[name] += 1

        queue: deque[str] = deque(
            name for name, deg in in_degree.items() if deg == 0
        )
        order: list[str] = []

        while queue:
            node = queue.popleft()
            order.append(node)
            for dependent in adjacency[node]:
                in_degree[dependent] -= 1
                if in_degree[dependent] == 0:
                    queue.append(dependent)

        if len(order) != len(self._capabilities):
            cycle_nodes = [
                name for name, deg in in_degree.items() if deg > 0
            ]
            raise CircularDependencyError(
                f"Circular dependency detected among: {cycle_nodes}"
            )

        return order

    def all_names(self) -> list[str]:
        return list(self._capabilities.keys())

# octopussy/core/lifecycle_manager.py
from __future__ import annotations

import asyncio
import logging
from typing import Any

from octopussy.core.registry import OctopussyCapabilityRegistry

logger = logging.getLogger(__name__)


class OctopussyCapabilityLifecycleManager:
    """
    Orchestrates all Capabilities through the mandatory six-phase CCA lifecycle:
    instantiate → initialize() → inject_dependency() → start()
        → [runtime] →
    stop() → cleanup()

    Startup: topological order (computed by Kahn's algorithm).
    Shutdown: strict reverse topological order.
    """

    def __init__(self, registry: OctopussyCapabilityRegistry) -> None:
        self._registry = registry
        self._order:   list[str] = []

    async def start_all(self) -> None:
        """
        Start all Capabilities in topological order.
        Each Capability goes through: initialize() → inject_dependency() → start()
        Raises CapabilityLifecycleError if any phase fails.
        """
        self._order = self._registry.topological_order()
        logger.info("Starting %d Capabilities in order: %s", len(self._order), self._order)

        for name in self._order:
            cap = self._registry.get(name)
            logger.info("Initializing Capability: %s", name)
            await cap.initialize()

        for name in self._order:
            cap = self._registry.get(name)
            logger.info("Injecting dependencies for Capability: %s", name)
            await cap.inject_dependency(self._registry)

        for name in self._order:
            cap = self._registry.get(name)
            logger.info("Starting Capability: %s", name)
            await cap.start()

        logger.info("All Capabilities started successfully.")

    async def stop_all(self) -> None:
        """
        Stop all Capabilities in strict reverse topological order.
        Each Capability goes through: stop() → cleanup()
        """
        reverse_order = list(reversed(self._order))
        logger.info("Stopping Capabilities in reverse order.")

        for name in reverse_order:
            cap = self._registry.get(name)
            logger.info("Stopping Capability: %s", name)
            try:
                await cap.stop()
            except Exception as exc:
                logger.error("Error stopping Capability %s: %s", name, exc)

        for name in reverse_order:
            cap = self._registry.get(name)
            logger.info("Cleaning up Capability: %s", name)
            try:
                await cap.cleanup()
            except Exception as exc:
                logger.error("Error cleaning up Capability %s: %s", name, exc)

        logger.info("All Capabilities stopped.")

15.4 Main Entry Point
# octopussy/main.py
from __future__ import annotations

import asyncio
import logging
import signal
import sys

from octopussy.core.lifecycle_manager import OctopussyCapabilityLifecycleManager
from octopussy.core.registry import OctopussyCapabilityRegistry
from octopussy.capabilities.registration import register_all_capabilities

logging.basicConfig(
    level   = logging.INFO,
    format  = "%(asctime)s %(levelname)s %(name)s %(message)s",
    stream  = sys.stdout,
)
logger = logging.getLogger(__name__)


async def main() -> None:
    """
    Octopussy platform entry point.
    1. Create registry and register all 26 Capabilities.
    2. Start all Capabilities in topological order.
    3. Wait for shutdown signal (SIGINT / SIGTERM).
    4. Stop all Capabilities in reverse topological order.
    """
    registry = OctopussyCapabilityRegistry()
    register_all_capabilities(registry)

    lifecycle_manager = OctopussyCapabilityLifecycleManager(registry)

    shutdown_event = asyncio.Event()

    def _signal_handler(sig: int, _frame: object) -> None:
        logger.info("Received signal %s; initiating graceful shutdown.", signal.Signals(sig).name)
        shutdown_event.set()

    loop = asyncio.get_running_loop()
    for sig in (signal.SIGINT, signal.SIGTERM):
        loop.add_signal_handler(sig, _signal_handler, sig, None)

    logger.info("Starting Octopussy v4.0.0...")
    await lifecycle_manager.start_all()
    logger.info("Octopussy v4.0.0 is running. Press Ctrl+C to stop.")

    await shutdown_event.wait()

    logger.info("Shutting down Octopussy v4.0.0...")
    await lifecycle_manager.stop_all()
    logger.info("Octopussy v4.0.0 shutdown complete.")


if __name__ == "__main__":
    asyncio.run(main())

from __future__ import annotations
import pytest
import time
from uuid import uuid4
from octopussy.core.enums import MessagePriority, MessageType
from octopussy.core.message import OctopussyMessage


SECRET = b"test-hmac-secret-32-bytes-long!!"


def make_message(**kwargs) -> OctopussyMessage:
    defaults = dict(
        sender_id    = uuid4(),
        recipient_id = uuid4(),
        message_type = MessageType.TASK,
        payload      = {"content": "test task"},
        secret       = SECRET,
        trace_id     = uuid4(),
        priority     = MessagePriority.NORMAL,
        ttl_seconds  = 300,
    )
    defaults.update(kwargs)
    return OctopussyMessage.create(**defaults)


class TestOctopussyMessage:

    def test_create_produces_valid_signature(self):
        msg = make_message()
        assert isinstance(msg.signature, bytes)
        assert len(msg.signature) > 0

    def test_verify_signature_succeeds_with_correct_secret(self):
        msg = make_message()
        assert msg.verify_signature(SECRET) is True

    def test_verify_signature_fails_with_wrong_secret(self):
        msg = make_message()
        assert msg.verify_signature(b"wrong-secret-32-bytes-long!!!!!") is False

    def test_message_is_frozen(self):
        from dataclasses import FrozenInstanceError
        msg = make_message()
        with pytest.raises(FrozenInstanceError):
            object.__setattr__(msg, "payload", {"evil": "mutation"})

    def test_expired_message_detected(self):
        msg = make_message(ttl_seconds=1)
        time.sleep(1.1)
        assert msg.is_expired() is True

    def test_not_expired_message(self):
        msg = make_message(ttl_seconds=300)
        assert msg.is_expired() is False

    def test_non_serializable_payload_raises(self):
        with pytest.raises(ValueError, match="JSON-serializable"):
            make_message(payload={"bad": object()})

    def test_zero_ttl_raises(self):

        assert sm.state == AgentState.INITIALIZING

    def test_terminal_state_has_no_transitions(self):

            sm.transition(AgentState.IDLE)

    def test_increment_within_limit_does_not_raise(self):
        guard = LoopGuard(max_iterations=5)
        for _ in range(5):
            guard.increment()

    def test_increment_beyond_limit_raises(self):
        guard = LoopGuard(max_iterations=3)
        for _ in range(3):
            guard.increment()
        with pytest.raises(LoopLimitExceededError):
            guard.increment()

    def test_reset_allows_new_cycle(self):
        guard = LoopGuard(max_iterations=2)
        guard.increment()
        guard.increment()
        guard.reset()
        guard.increment()  # should not raise

    def test_zero_max_iterations_raises_on_init(self):
        with pytest.raises(ValueError):
            LoopGuard(max_iterations=0)

    def test_iteration_count_tracked(self):
        guard = LoopGuard(max_iterations=10)
        guard.increment()
        guard.increment()
        assert guard.iteration == 2

from __future__ import annotations
import pytest
from octopussy.core.registry import OctopussyCapabilityRegistry
from octopussy.core.exceptions.octopussy_exceptions import (
    CircularDependencyError,
    CapabilityNotFoundError,

class MockCapability:
    def __init__(self, name: str):
        self.name = name


class TestOctopussyCapabilityRegistry:

    def test_register_and_get(self):
        registry = OctopussyCapabilityRegistry()
        cap = MockCapability("A")
        registry.register("A", cap, [])
        assert registry.get("A") is cap

    def test_get_unknown_raises(self):
        registry = OctopussyCapabilityRegistry()
        with pytest.raises(CapabilityNotFoundError):
            registry.get("NonExistent")

    def test_circular_dependency_detected(self):
        registry = OctopussyCapabilityRegistry()
        registry.register("A", MockCapability("A"), [])
        registry.register("B", MockCapability("B"), ["A"])
        with pytest.raises(CircularDependencyError):
            registry.register("A2", MockCapability("A2"), ["B"])
            # This would create A → B → A2 → A cycle if A2 depends on B and B depends on A
            # Simpler: register C depending on B, then make B depend on C
            registry2 = OctopussyCapabilityRegistry()
            registry2.register("X", MockCapability("X"), [])
            registry2.register("Y", MockCapability("Y"), ["X"])
            # Now try to make X depend on Y — cycle
            # Can't re-register X, so test via direct cycle:
            registry3 = OctopussyCapabilityRegistry()
            registry3.register("P", MockCapability("P"), ["Q"])  # Q not yet registered
            # This should raise CapabilityNotFoundError on topological_order

    def test_topological_order_simple_chain(self):
        registry = OctopussyCapabilityRegistry()
        registry.register("A", MockCapability("A"), [])
        registry.register("B", MockCapability("B"), ["A"])
        registry.register("C", MockCapability("C"), ["B"])
        order = registry.topological_order()
        assert order.index("A") < order.index("B")
        assert order.index("B") < order.index("C")

    def test_duplicate_registration_raises(self):
        registry = OctopussyCapabilityRegistry()
        registry.register("A", MockCapability("A"), [])
        with pytest.raises(ValueError, match="already registered"):
            registry.register("A", MockCapability("A2"), [])

18. EVOLUTION &amp; VERSIONING POLICY
18.1 Semantic Versioning
Octopussy follows Semantic Versioning 2.0.0 (MAJOR.MINOR.PATCH):

MAJOR: Breaking changes to any Capability Contract or OctopussyMessage format.
MINOR: New Capabilities, new plugin types, new configuration fields (backward-compatible).
PATCH: Bug fixes, performance improvements, security patches.

18.2 Capability Evolution Envelope
Each Capability carries an Evolution Envelope:
# Example: LLMRouterCapability Evolution Envelope
CAPABILITY_VERSION     = "4.0"
MINIMUM_CLIENT_VERSION = "4.0"
DEPRECATION_POLICY     = "12 months notice before removal of any Contract method"
MIGRATION_GUIDE        = "docs/migration/llm_router_4_0.md"

18.3 Configuration Schema Versioning
All configuration files carry schema_version: "4.0". When the schema changes:

New schema version is introduced with full backward compatibility for one major version.
ConfigurationCapability validates schema version and reports clear migration errors.
Migration guide is published before the old schema version is removed.

18.4 MCP Protocol Version
Octopussy 4.0.0 targets MCP protocol version 2025-11-25. The MCPClientAdapter sends MCP-Protocol-Version: 2025-11-25 on all requests. The OctopussyMCPServer validates the protocol version header and returns a clear error for unsupported versions.
